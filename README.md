# xcapselx

I help builders make GitHub repositories easier to run, test, and maintain.

## GitHub Action

Use my repo health check action in your CI:

```yaml
- uses: xcapselx/repo-health-checks@v1
  with:
    path: '.'
    fail-on-low-score: 'false'
```

Scans README, LICENSE, SECURITY.md, dependency lockfiles, and file tree hygiene. No network calls, no data uploads.
